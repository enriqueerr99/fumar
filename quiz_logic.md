## CÁLCULO FINAL

**Después de las 13 preguntas:**

1. **Suma total de puntos** para cada perfil
2. **Perfil ganador** = el que tiene más puntos
3. **Tie-breaker:** Si hay empate entre dos perfiles, la Pregunta 5 (motivación explícita) tiene peso extra

**Ejemplo:**
- Control: 18 puntos
- Familia: 8 puntos
- Racional: 12 puntos
- Ansioso: 9 puntos

→ **Resultado: El Controlador** (18 puntos)

## NOTAS TÉCNICAS

- **Sistema:** Suma acumulativa de puntos
- **Escalabilidad:** Si una pregunta no se responde, suma 0
- **Múltiples respuestas (Pregunta 5):** Cada selección suma puntos independientemente
- **Penalizaciones:** Algunos ítems tienen puntos negativos (ej: -1) para descartar un perfil
- **Sensibilidad:** El sistema es lo suficientemente sensible para detectar matices, pero lo suficientemente robusto para dar resultados claros
